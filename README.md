# RU

### Что это?

Нет лицензии майнкрафт, а хочется пользоваться всеми удобствами ATLauncher? Тогда ты по адресу!

`ATLCrack` - это проект с открытым исходным кодом целью которого было **безопасно** работать ATLauncher **без лицензии** майнкрафт.

### Как использовать?

Всё просто!
1. Запускаешь `start.bat` / `start.sh` / `ATLCrack.jar`
2. Немного ждёшь скачивания оригинала ATL
3. Вводишь свой ник
4. Радуешься жизни!

[Видео с наглядной демонстрацией](https://www.youtube.com/watch?v=pYYzYiKed6g).

# EN

### What is this?

Don't have a Minecraft license but want to enjoy all the features of ATLauncher? Then you've come to the right place!

`ATLCrack` - is a open source project designed to allow ATLauncher to run securely without a Minecraft license.

### How to use?

It's simple
1. Run `start.bat` / `start.sh` / `ATLCrack.jar`
2. Wait a bit for the original ATL to download
3. Enter your username
4. Enjoy!

[Video with visual demonstration](https://www.youtube.com/watch?v=pYYzYiKed6g).
